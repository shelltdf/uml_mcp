# 存储与文件格式（广义数据库）

## 文件即主存储

| 扩展名 / 文件名 | 用途 |
|-----------------|------|
| `*.uml.md` | **一文件一图**；每文件通常一个带 `mermaid`（或约定 `uml`）的 fenced 块；多图用多个 `*.uml.md` 同目录存放 |
| `*.class.md` | **一文件一类**；类定义：推荐表格或列表字段（类名、继承、成员、可见性）；脚手架默认 **helloworld** |
| `*.code.md` | 非类**抽象契约**：全局函数、变量、宏等（表格/条目，非具体语言源码）；**路径上**置于各 `namespace_root` 下（与 `*.class.md` 同树），非各 `code_impls` 翻译单元目录 |
| `uml.sync.md` | 同步清单：命名空间目录、UML 根、代码根、规则文本与可选 YAML |

## 版本与兼容

- 本仓库**不**将上述格式定义为需自有 `idl/` 版本链的专有二进制格式；契约以 `02-physical/uml-vue-sdi/spec.md` 与示例文件为准。
