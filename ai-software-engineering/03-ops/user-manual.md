# 用户说明（uml-vue-sdi）

## 启动

1. 在 `uml-vue-sdi/` 执行 `npm install` 与 `npm run dev`。
2. 浏览器访问终端提示的本地 URL（通常为 `http://127.0.0.1:5173`）。

## 顶区布局

- **标题条**（`#title-strip`）单独一行：应用图标（`favicon.svg`）+ 名称；`document.title` 在存在未保存更改时加 **`·`** 后缀。
- **菜单栏**（`#menu-bar`）在其**下方**，从左到右：**文件**、**语言**（默认 **中文 / 英文** 两项，非循环）、**主题**（跟随系统 / 浅色 / 深色分项，非循环）、**窗口**（显示「文本内容」「属性」停靠区）、**帮助**（第一项为帮助信息 **F1**、文件格式说明、关于）。
- 点击顶层菜单展开下拉，**不与标题混在同一行**。

## 主区与右侧 Dock

- **中央**：已打开文件的标签行 + 下方画布（UML 预览、同步配置表单、类/代码说明等）。
- **右侧**：**Dock 列** — 靠内为 **Dock View**（可同时显示 **属性** 与 **文本内容** 编辑器），最右缘为 **Dock 按钮条**（切换各停靠窗口显示/隐藏）。
- **分割条**：主区与 Dock 列之间的 **竖条** 可左右拖动改变 Dock 宽度；**属性** 与 **文本内容** 同时展开时，中间的 **横条** 可上下拖动调整两区高度比例。
- 窄屏下布局可能变为上下叠放，仍以界面实际表现为准（见 `02-physical/uml-vue-sdi/spec.md`）。

## 快捷键与脏文档

- **F1**：打开帮助面板（浏览器内 `preventDefault`）。
- **未保存**：标签 `·`、状态栏提示、`beforeunload`、关闭标签与重置示例前确认。

## SDI 行为

- **单窗口**：一个主界面；多个文件以**顶部标签**切换。
- **未保存**：修改后标签显示脏标记（`·` 或等价样式）；刷新页面前浏览器可能提示（`beforeunload`）。

## 文件类型

- `*.uml.md`：编辑 Markdown，右侧/下方预览 Mermaid。
- `*.class.md` / `*.code.md`：文本编辑；预览区可显示提示「非 UML 预览」。二者均建议放在各 **`namespace_root`** 目录树内；`*.code.md` **不**放在 `code_impls` 所指的真实代码根。
- `uml.sync.md`：查看/编辑同步配置 YAML 与规则正文；画布内为表单（路径仅写入文档；命名空间根为单一路径；代码实现通过弹窗添加；规则正文区随内容增高）。详见 `02-physical/uml-vue-sdi/spec.md`。

## 悬停快捷键

- 工具条按钮若绑定快捷键，应在 `title` 中展示；无快捷键时标注「无全局快捷键」。
